# Monotonic

> a sequence or function is monotonic if it maintains a **consistent trend in one direction** -- either non-decreasing or non-increasing throughout 

---

This can be useful for problems where we can use [[binary-search]] 

> The array has a monotonic properties = parts of the array follow a predictable one-directional pattern
